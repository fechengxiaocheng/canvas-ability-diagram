# 基于canvas的react能力图组建

    git clone git@github.com:fechengxiaocheng/canvas-ability-diagram.git
    npm install
    npm start

## 组建目录

-src/components/Libility.js

## 使用方法

    // 在业务代码中引入

    import Libility from './components/Libility';
    <Libility value={[['颜值', 0.9], ['身材', 0.2], ['才华', 0.5], ['性格', 0.5], ['品行', 0.5], ['才艺', 0.5]]}/>

## 主要功能点

  * 任意多边形展示自适应
  * 移动端自适应
